⚽ Football Tactical Analysis System (FTAS)A Computer Vision-based system designed to analyze football match tactics from video footage, specifically optimized for V-League and student tournament data.🚀 Key FeaturesObject Detection & Tracking: Real-time player, ball, and referee detection using YOLOv8 and ByteTrack.Team Classification: Automatic jersey color recognition using K-Means clustering.Tactical Metrics:Compact Score: Measures squad cohesion and connectivity over time.Pressing Intensity: Calculates player velocity and pressure radius around the ball.Formation Adherence: Automatically identifies tactical setups (e.g., 4-4-2, 4-3-3).Automated Insights: Generates natural language tactical summaries based on calculated data.Web Dashboard: Integrated FastAPI backend to visualize heatmaps and passing networks.🛠 Tech StackAI/CV: Python, PyTorch, OpenCV, YOLOv8, ByteTrack.Backend: FastAPI.Frontend: HTML/CSS/JS with Matplotlib/Plotly for visualization.📅 Development RoadmapPhase 1 (Week 1-3): Environment setup and basic CV pipeline.Phase 2 (Week 4-7): Development of tactical analysis algorithms.Phase 3 (Week 8-10): Web system integration and data visualization.Phase 4 (Week 11-13): Testing, evaluation (mAP, Precision, Recall), and final report.📊 EvaluationThe system is tested on 3–5 real-world match videos, focusing on addressing camera angle challenges specific to Vietnamese stadiums.
